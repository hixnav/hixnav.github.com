安装

获取HixNav

获取windows版本，点击下载

获取Linux版本，点击下载

查看源码，请访问仓库HixNav

配置

你需要一个Mysql的数据库，以及一个对象存储服务，这些服务可以从云服务厂商那里获取。

配置系统环境变量
# mysql
# DNS={user}:{password}@tcp({host}:{port})/haixin?charset=utf8mb4&parseTime=True&loc=Local
DNS

# tecentCOS
# COS=https://%s-{your}.myqcloud.com/
COS 
COSSecretID
COSSecretKey

运行

$ hixnav

访问

系统默认使用80端口
